# 💬 Local Web Chat — Навчальний проєкт

Легкий веб-чат у реальному часі, створений як навчальний проєкт для вивчення хмарних технологій, WebSocket-комунікації та клієнт-серверної архітектури.

Проєкт реалізований на базі **Flask** з використанням **Flask-SocketIO** для real-time взаємодії між клієнтами.

---

## 🚀 Можливості

* ✅ Надсилання повідомлень у реальному часі
* ✅ Збереження історії чату
* ✅ Завантаження та відображення зображень (jpg, png)
* ✅ Видалення повідомлень
* ✅ Нікнейми користувачів (збереження у localStorage)
* ✅ Автоматичне кольорове маркування користувачів
* ✅ Підтримка emoji

---

## 🛠 Технології

* Backend: **Flask**
* Real-time комунікація: **Flask-SocketIO**
* Протокол клієнтської взаємодії: **Socket.IO**
* База даних: **SQLite**
* Frontend: HTML, CSS, JavaScript

---

## 📂 Структура проєкту

```
.
├── app.py              # Основний Flask застосунок
├── chat.db             # SQLite база даних
├── uploads/            # Завантажені зображення
└── templates/
    └── index.html      # Frontend частина
```

## ⚙️ Встановлення та запуск

### 1️⃣ Клонування репозиторію

```bash
git clone <your-repo-url>
cd <project-folder>
```

### 2️⃣ Встановлення залежностей

```bash
pip install flask flask-socketio
```

### 3️⃣ Запуск застосунку

```bash
python app.py
```

Застосунок буде доступний за адресою:

```
http://localhost:5000
```

---

## 🗄 Архітектура

### 🔹 Backend

* Flask обробляє HTTP-запити
* SocketIO забезпечує WebSocket-комунікацію
* SQLite зберігає історію повідомлень
* Файли зображень зберігаються локально в папці `uploads`

### 🔹 Frontend

* Підключення до Socket.IO серверу
* Динамічне оновлення DOM без перезавантаження сторінки
* Відправка тексту та зображень через API
